# 新楓之谷 現金道具期望值計算機

計算新楓之谷現金道具的期望值與投資報酬率。

## 功能

### 新年氣息
- 計算投入金額可獲得的期望氣息數量
- 根據貪心算法計算可湊成的心願箱數量（超越 > 大吉 > 中吉 > 小吉）
- 計算期望總價值與報酬率

### 星光錦囊
- 計算第一階段道具的期望獲得數量
- 支援自訂各道具市場價值
- 計算期望總價值與報酬率

https://chiao75543.github.io/MSCashItemExpected/


## 購買方式計算

| 方式 | 計算公式 |
|------|----------|
| 點卡儲值 | 投入金額 ÷ 折數 |
| 讀卡機 | 投入金額 × 1.05 |
| 原價 | 投入金額 |
| 送禮 | 投入金額 ÷ 折數 |
